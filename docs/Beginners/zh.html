入门指南

Summary

这一系列的文章的宗旨是让初学者们能够具备所需的技能、充满信心地踏上成长为一个网页设计师或者开发者的旅程。

准备

在学习过程中，你将会编写很多的代码并在网页浏览器中进行测试。在此之前，你需要先安装下面这些程序：

  • 最新版本的浏览器。如果你还没有安装它们，就从它们主页下载吧：

    • 全平台
      • Chrome
      • Firefox
      • Opera
    • Windows
      • Internet Explorer
    • OS X
      • Safari.

  • 实用的文本编辑器。这里有一些不错的选择：

    • 全平台
      • Brackets
      • LightTable
      • Bluefish
    • Windows
      • Notepad++
      • Hbulider
    • OS X
      • Textmate
      • TextWrangler
      • Atom
    • Linux 发行版
      • Geany
      • gedit
