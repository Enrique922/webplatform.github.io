Desarrollo web para principiantes

Resumen

Este conjunto de artículos es para que los principiantes los lean con confianza, comenzando a construir sus habilidades fundamentales desde el inicio de su viaje para convertirse en un diseñador o desarrollador web.

Antes de comenzar

En este curso escribirás una gran cantidad de código y lo pondrás a prueba en los navegadores web. Antes de comenzar debes tener instalado por lo menos lo siguiente:

  • Los más nuevos navegadores que puedas instalar. Si acaso no los tienes ya, descárgalos desde las páginas de Chrome, Firefox, Internet Explorer, Opera y Safari.
  • Un editor de texto decente. Las opciones gratuitas razonables son Notepad++ para Windows, Text Wrangler para Mac, Bluefish para Linux.

Ahora, arremángate la camisa y preparémonos a aprender. ¡Explora nuestra lista completa de artículos didácticos si terminas la lista de esta página y deseas aprender aún más!
