SVG

Resumen

Imágenes vectoriales escalables o Scalable Vector Graphics (SVG) es un lenguaje de etiquetado XML para describir estructuralmente imágenes vectoriales bidimensionales. En esencia, SVG es a las imágenes lo que el HTML es al texto. SVG es una recomendación del W3C y está explícitamente diseñado para trabajar con otros estándares del W3C como el HTML, las CSS o el DOM.

Antecedentes

SVG se elaboró ​​por primera vez en 1998, en un intento por crear un lenguaje estandarizado de etiquetado de imágenes vectoriales a ser usado por los navegadores (tecnologías propietarias para ese momento incluían VML de Microsoft y PGML de Sun/Adobe.)

  • Primera versión: La primera versión estandarizada fue SVG 1.0. La recomendación SVG 1.0 se publicó en el 2001.

  • Versión estable actual: SVG 1.1 es la implementación estable actual soportada en todos los navegadores modernos y publicada por primera vez en el 2003. También hay algunos vástagos de SVG con diferentes propósitos, como SVG Tiny 1.2, pero para referencia debes ceñirte al SVG 1.1.

  • Trabajo actual y futuro: El desarrollo de SVG 1.2 ha sido abandonado en favor de la próxima especificación SVG 2.0.
