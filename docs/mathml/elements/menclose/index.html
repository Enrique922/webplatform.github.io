menclose

Summary

The MathML menclose element renders its content inside an enclosing notation specified by the notation attribute.

Overview Table

DOM Interface
mathml

Examples

This example shows a simple markup using menclose:



<math>
  <menclose notation="circle box">
    <mi> x </mi>
    <mo> + </mo>
    <mi> y </mi>
  </menclose>
</math>
