背景

层叠样式表用于定义（造型，布局，动画）和标记内容（如HTML、XHTML、SVG 或 XML）的语言。它是开放化 Web 平台的组成部分之一。

CSS 允许你通过一系列选择器（selectors）和属性（properties）来为内容设定样式。

  • 第一版: CSS 1 建议由 Håkon Wium Lie 和 Bert Bos 完成，并于1996年问世。
  • 当前稳定版: CSS 2 建议诞生于1998年，尽管很多年都保持稳定，但直到2011年才正式完成。CSS3 的一些模块自（2012年12月）发布以来也很稳定。参见“进展和展望”。
  • 进展及展望: CSS 工作组正在研究制定大量的“模块”，它们描述了 CSS3 这一技术集合的方方面面。某些模块很稳定，另一些则处于实验阶段，缺乏良好的跨浏览器支持。甚至很多 CSS4 模块也初露端倪，尽管将其投入正式使用仍需假以时日。更多关于模块/规范的进度信息参见 CSS 工作组工作动态页面。

助力 CSS 技术

CSS 的开放工作由 CSS 工作组主持。欲知详情，请前往CSS 工作组主页。若对参与规范制定有反馈和疑问，www-style 邮件列表是个好去处。切记，不要在这里咨询诸如“我怎么用 CSS 给网页指定样式”这类问题。
