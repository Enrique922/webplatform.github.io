HTML elements & attributes

Presently, there is no structured, semantic way to associate an HTML element element with the attributes that define that element, such as the Applies to form interface provided for API documentation, where members (events, methods, and properties) are associated with the object via the Applies to field. For example, the anchor element page has no information about the attributes that apply to it. Furthermore, the HTML elements have no mechanism that associates them with the events and methods that pertain to them.

Tasks

  • Developing a cross-referencing system that associates each of the some 200 HTML attributes with any of the 155 elements to which it belongs.
  • Providing the DOM APIs with an Applies to field for the HTML elements upon which they operate.
  • Manually populating the fields for all artifacts.
