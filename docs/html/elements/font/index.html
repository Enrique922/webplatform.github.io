font

Summary

The font element is an obsolete tag that was used to specify the typeface, size, and color of the text it contained.

Overview Table

DOM Interface
HTMLFontElement

This element was deprecated in HTML 4.01, and became obsolete in HTML5. It has no semantic meaning. For identical functionality, use a span tag, and **use CSS instead**.
