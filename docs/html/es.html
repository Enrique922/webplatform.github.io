HTML

Resumen

Hypertext Markup Language (HTML) es el lenguaje de etiquetado para las páginas web. El World Wide Web Consortium mantiene el estándar del HTML.

Antecedentes

HTML es el lenguaje con el que estructuramos nuestros contenidos en la World Wide Web, uno de los principales componentes de la Plataforma Web Abierta.

  • Primera versión: La primera versión del HTML fue descrita por Tim Berners-Lee, el inventor de la Web, a finales de 1991.
  • La versión estable actual: La Recomendación de HTML 4.01 se publicó en diciembre de 1999.
  • El trabajo actual y futuro: El borrador más estable se publica en el Borrador de trabajo del HTML5 del W3C y también hay un estándar vivo del HTML, que contiene las últimas ideas y características experimentales, publicado por el WHATWG.

Contribuir con la tecnología HTML

El desarrollo del HTML ha sido conducido por diversas organizaciones durante su historia, incluyendo al CERN, IETF, W3C… El desarrollo se ha llevado a cabo principalmente en las listas de correo del Grupo de trabajo de HTML. Este es un buen lugar para comenzar si quieres sugerir nuevas características o enviar comentarios. Para obtener noticias e información más general, consulta la página principal del HTML en el W3C.

El W3C también está solicitando comentarios de revisión sobre la Lista de Comprobación de Accesibilidad de Medios
